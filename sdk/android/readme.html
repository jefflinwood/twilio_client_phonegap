Twilio Android Client SDK README

Ahoy hoy and welcome to Twilio Client for Android! In just a few minutes you'll be adding telephony features to your Android app.

We recommend going through the Quickstart Guide to get yourself familiar with the SDK. And you'll want to get yourself a Twilio account by heading to https://www.twilio.com/try-twilio if you don't have one already.

Here's an overview of the SDK contents:

  • ChangeLog - a list of changes since the last release
  • readme - this file
  • Quickstart Guide - a guide with sample Eclipse projects that take you step-by-step through building an Android application to connect to Twilio
  • FAQ - frequently asked questions
  • Javadocs - Standard Javadoc-style documentation for the SDK
  • Resources/ - sound resources used with Twilio Client Android
  • BasicPhone/ - sample app that shows you Twilio Client in action
  • helper-libs/ - server-side helper libraries to use in conjunction with your Android application
  • libs/ - the Twilio Client SDK JAR file as well as native libraries that need to be copied into your project
  • Server/ - sample server backend - https://github.com/twilio/mobile-quickstart

Happy coding!
-Team Twilio
